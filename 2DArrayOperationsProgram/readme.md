# 2D Array Operations Program

This Java program provides a menu-driven interface to perform various operations on a 2D array, including calculating the sum of diagonal elements, upper diagonal elements, and lower diagonal elements.

## Features

- **Sum of Diagonal Elements:** Computes the sum of the main diagonal elements of the 2D array.
- **Sum of Upper Diagonal Elements:** Calculates the sum of the elements above the main diagonal.
- **Sum of Lower Diagonal Elements:** Calculates the sum of the elements below the main diagonal.
- **Exit:** Allows the user to exit the program.
